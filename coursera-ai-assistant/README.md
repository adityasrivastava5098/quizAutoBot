# 🎓 Coursera AI Learning Assistant

A Chrome extension that uses Claude AI to help you understand and answer Coursera quiz questions.

## Features
- 🔍 **Scan** — Automatically extracts questions from Coursera quiz pages
- 🤖 **Solve** — Sends questions to Claude AI for explanations
- 🖊 **Highlight** — Visually highlights suggested answers on the page
- 3 Modes: **Explain** (learn why), **Hint** (guided learning), **Full Answer**

## Installation (Chrome)

1. Go to `chrome://extensions/`
2. Enable **Developer Mode** (top right toggle)
3. Click **"Load unpacked"**
4. Select the `coursera-ai-assistant` folder
5. The extension icon will appear in your toolbar

## Setup

1. Click the extension icon
2. Enter your **Anthropic API key** (get one at https://console.anthropic.com)
3. Click **Save Key**

## Usage

1. Navigate to a Coursera quiz or assignment page
2. Click the extension icon
3. Click **Scan Page for Questions**
4. Choose your mode (Explain / Hint / Answer)
5. Click **Solve with AI**
6. Click **Highlight Answers** to see suggestions on the page

## Getting an API Key

1. Go to https://console.anthropic.com
2. Sign up / log in
3. Go to **API Keys** → **Create Key**
4. Copy and paste into the extension

## Icons

The `icons/` folder needs 3 PNG files: `icon16.png`, `icon48.png`, `icon128.png`
You can use any 🎓 icon exported at those sizes, or generate them free at https://favicon.io
